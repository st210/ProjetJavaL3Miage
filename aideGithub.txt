UTILISATION GIT:

-- Initilisation
git remote add origin https://github.com/<utilisateur>/<projet> (définir repo github comme original)
git remote -v (s'assurer qu'il le connait bien)

-- Commit
git status
git add <fichier>
git commit -m "<message commit>"

-- Branches
git branch (afficher les branches)
git branch <nomNouvelleBranche> (créer branche) & git checkout <nomBranche> (se déplacer dans branche)
ou
git checkout -b <nomNouvelleBranche>
git checkout <brancheA> & git merge <brancheB> (merge 2 branches, il faut être à l'intérieur de la A pour merger la B dedans)

-- Lien avec Github
git pull
git push

-- Modifications
git blame <nomFichier> (voir qui a fait derniere modif)
git show <numeroDuBlame> (voir le message/pourquoi cette modif a été faite)
git stash (mettre en attente les modifs en cours)
git stash pop (les récupérer) & git stash apply (si garder modifs dans stash tout en modifiant)
